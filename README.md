# **Exercicio 3**
  * Fazer um programa para ler quatro valores inteiros A, B, C e D. A seguir, calcule e mostre a diferença do produto
de A e B pelo produto de C e D segundo a fórmula:
  * DIFERENCA = (A * B - C * D).

## Tabela de Dados

| **Entrada**  | **Saida**    | 
|----------|------------------
| 5 6 7 8   |DIFERENCA = -26 |
| 5 6-7 8   |DIFERENCA = 86  |
